# Thor Stream Butler v0.3.0-alpha.1 — VPN-Aware Preview / VPN-bewusste Vorschau

## English

### Highlights

- Network diagnostics now identify an active VPN before its underlying Wi-Fi,
  Ethernet, or cellular transport.
- VPN routing receives a clear, non-critical quality warning and a practical
  recommendation when game streaming is unstable.
- The complete app interface remains available in English and German.
- Still no account, cloud backend, advertising, tracking, or telemetry.

### Changes

- Adds a pure, unit-tested connection-type resolver with explicit VPN priority.
- Treats an otherwise healthy VPN connection as usable (yellow), not
  problematic. Measured latency, jitter, packet loss, internet validation, and
  host reachability still determine whether the result becomes red.
- Improves the Release workflow: prerelease tags may build an explicitly named
  debug APK when production-signing secrets are absent, while stable tags remain
  blocked without complete signing configuration.
- Validates that release notes contain equivalent English and German sections,
  derives the bilingual title from this file, and marks the GitHub release as a
  prerelease automatically.

### Fixes

- An active VPN can no longer be hidden behind a simultaneously reported Wi-Fi,
  Ethernet, or cellular transport in the network-test result.

### Known limitations

- **Development signature:** The APK is signed with a GitHub Actions Android
  debug certificate. It is a prerelease for testing, not a production-signed
  stable build.
- **Update path:** This APK cannot reliably update v0.1.0-alpha.1,
  v0.2.0-alpha.1, or another debug build. Uninstall an older alpha first;
  Android then removes its local settings, configured hosts, and measurement
  history.
- **Hardware validation:** Unit tests, lint, APK builds, package metadata,
  signature, and checksum are verified. No physical Android handheld was
  connected for installation during this release run.
- Automatic Sunshine and Moonlight host discovery is not implemented yet.
- Website screenshots are sanitized interface previews, not real-device captures.
- Some Wi-Fi and network values may be unavailable because of Android version,
  device support, permissions, VPN implementation, or platform privacy
  restrictions. The app marks unavailable values instead of inventing data.

### Installation or update

1. Uninstall any previously installed Thor Stream Butler alpha. This removes
   the previous alpha's local data because configuration export is not available yet.
2. Download `ThorStreamButler-v0.3.0-alpha.1-debug.apk` and its `.sha256` file
   from the assets below.
3. Verify the SHA-256 checksum.
4. Allow installation from the browser or file manager you use and install the
   APK on Android 9 or newer.

Package name: `de.thorstream.butler`

Minimum Android version: Android 9 (API 28)

### Privacy note

Thor Stream Butler contains no advertising, tracking, or telemetry. Network
measurements and configured hosts stay local on the device. Release files and
public screenshots contain no personal network data.

## Deutsch

### Highlights

- Die Netzwerkdiagnose erkennt ein aktives VPN jetzt vor dem darunterliegenden
  WLAN-, Ethernet- oder Mobilfunktransport.
- VPN-Routing erhält einen klaren, nicht kritischen Qualitätshinweis und bei
  instabilem Gaming-Streaming eine praktische Empfehlung.
- Die vollständige App-Oberfläche bleibt auf Englisch und Deutsch verfügbar.
- Weiterhin kein Konto, kein Cloud-Backend, keine Werbung, kein Tracking und
  keine Telemetrie.

### Änderungen

- Ergänzt eine reine, unit-getestete Auflösung des Verbindungstyps mit
  ausdrücklichem VPN-Vorrang.
- Bewertet eine ansonsten gesunde VPN-Verbindung als nutzbar (Gelb), nicht als
  problematisch. Gemessene Latenz, Jitter, Paketverlust, Internetvalidierung und
  Host-Erreichbarkeit entscheiden weiterhin, ob das Ergebnis Rot wird.
- Verbessert den Release-Workflow: Prerelease-Tags dürfen bei fehlenden
  produktiven Signing-Secrets eine eindeutig benannte Debug-APK bauen, während
  stabile Tags ohne vollständige Signing-Konfiguration blockiert bleiben.
- Prüft, dass Release Notes gleichwertige englische und deutsche Abschnitte
  enthalten, übernimmt den zweisprachigen Titel aus dieser Datei und kennzeichnet
  das GitHub-Release automatisch als Prerelease.

### Fehlerbehebungen

- Ein aktives VPN kann im Netzwerktest nicht mehr hinter einem gleichzeitig
  gemeldeten WLAN-, Ethernet- oder Mobilfunktransport verborgen bleiben.

### Bekannte Einschränkungen

- **Entwicklungssignatur:** Die APK ist mit einem Android-Debug-Zertifikat von
  GitHub Actions signiert. Sie ist ein Prerelease zum Testen und kein produktiv
  signierter stabiler Build.
- **Update-Pfad:** Diese APK kann v0.1.0-alpha.1, v0.2.0-alpha.1 oder einen
  anderen Debug-Build nicht zuverlässig aktualisieren. Deinstalliere eine ältere
  Alpha zuerst; Android entfernt dabei deren lokale Einstellungen, konfigurierte
  Hosts und Messhistorie.
- **Hardware-Prüfung:** Unit Tests, Lint, APK-Builds, Paketmetadaten, Signatur und
  Prüfsumme werden geprüft. Während dieses Release-Durchlaufs war kein physischer
  Android-Handheld für eine Installation verbunden.
- Die automatische Sunshine- und Moonlight-Host-Erkennung ist noch nicht umgesetzt.
- Website-Screenshots sind bereinigte Oberflächenvorschauen und keine echten
  Geräteaufnahmen.
- Einige WLAN- und Netzwerkwerte können wegen Android-Version,
  Geräteunterstützung, Berechtigungen, VPN-Implementierung oder
  Datenschutzbeschränkungen der Plattform nicht verfügbar sein. Die App
  kennzeichnet nicht verfügbare Werte, statt Daten zu erfinden.

### Installation oder Update

1. Deinstalliere eine zuvor installierte Thor-Stream-Butler-Alpha. Dadurch werden
   deren lokale Daten entfernt, da ein Konfigurationsexport noch nicht verfügbar ist.
2. Lade `ThorStreamButler-v0.3.0-alpha.1-debug.apk` und die zugehörige
   `.sha256`-Datei aus den Dateien unten herunter.
3. Prüfe die SHA-256-Prüfsumme.
4. Erlaube die Installation aus dem verwendeten Browser oder Dateimanager und
   installiere die APK auf Android 9 oder neuer.

Paketname: `de.thorstream.butler`

Minimale Android-Version: Android 9 (API 28)

### Datenschutzhinweis

Thor Stream Butler enthält keine Werbung, kein Tracking und keine Telemetrie.
Netzwerkmessungen und konfigurierte Hosts bleiben lokal auf dem Gerät.
Release-Dateien und öffentliche Screenshots enthalten keine persönlichen
Netzwerkdaten.
