# Thor Stream Butler v0.1.0-alpha.1 — First Preview / Erste Vorschau

## English

### Highlights

- First installable development preview for Android 9 and newer.
- Controller-first dashboard for installed streaming apps and custom Android apps.
- Local network diagnostics, quality rating, host management, Wake-on-LAN,
  settings, and measurement history.
- No account, cloud backend, advertising, tracking, or telemetry.

### Changes

- Adds the complete initial MVP described in the project README.
- Adds responsive portrait and landscape layouts with D-pad, controller, keyboard,
  and touch operation.
- Adds local-only persistence with Room and DataStore.
- Adds English and German project pages, public documentation, and release information.

### Fixes

- This is the first public preview; there is no previous public APK to compare.

### Known limitations

- **Development signature:** The APK is signed with the standard Android debug
  certificate. It is a prerelease for testing, not a production-signed stable build.
- **Future update:** A production-signed stable version cannot update this APK.
  Uninstall this preview first; Android will then remove its local settings,
  configured hosts, and measurement history.
- **Language:** The in-app interface is currently German only. GitHub documentation,
  the project website, and these release notes are available in English and German.
- **Hardware validation:** The APK build, package metadata, and signature were
  verified. No physical Android handheld was connected for installation during
  this release run.
- Automatic Sunshine and Moonlight host discovery is not implemented yet.
- Website screenshots are sanitized interface previews, not real-device captures.
- Some Wi-Fi and network values may be unavailable because of Android version,
  device support, permissions, or platform privacy restrictions. The app marks
  unavailable values instead of inventing measurements.

### Installation or update

1. Download `ThorStreamButler-v0.1.0-alpha.1-debug.apk` from the assets below.
2. Verify it with `ThorStreamButler-v0.1.0-alpha.1-debug.apk.sha256`.
3. Allow installation from the browser or file manager you use, then install the APK.
4. Before installing a future production-signed stable release, uninstall this alpha.
   This removes the alpha's local app data because export is not available yet.

Package name: `de.thorstream.butler`

Minimum Android version: Android 9 (API 28)

### Privacy note

Thor Stream Butler contains no advertising, tracking, or telemetry. Network
measurements and configured hosts stay local on the device. Release assets and
public screenshots contain no personal network data.

## Deutsch

### Highlights

- Erste installierbare Entwicklungsvorschau für Android 9 und neuer.
- Controller-first Dashboard für installierte Streaming-Apps und frei gewählte Android-Apps.
- Lokale Netzwerkdiagnose, Qualitätsbewertung, Hostverwaltung, Wake-on-LAN,
  Einstellungen und Messhistorie.
- Kein Konto, Cloud-Backend, Werbung, Tracking oder Telemetrie.

### Änderungen

- Fügt den vollständigen ersten MVP aus der Projekt-README hinzu.
- Fügt responsive Portrait- und Landscape-Layouts mit D-Pad-, Controller-, Tastatur-
  und Touch-Bedienung hinzu.
- Fügt ausschließlich lokale Speicherung mit Room und DataStore hinzu.
- Fügt englische und deutsche Projektseiten, öffentliche Dokumentation und
  Release-Informationen hinzu.

### Fehlerbehebungen

- Dies ist die erste öffentliche Vorschau; es gibt keine vorherige öffentliche APK
  zum Vergleichen.

### Bekannte Einschränkungen

- **Entwicklungssignatur:** Die APK ist mit dem standardmäßigen Android-Debug-Zertifikat
  signiert. Sie ist ein Prerelease zum Testen und kein produktiv signierter stabiler Build.
- **Zukünftiges Update:** Eine produktiv signierte stabile Version kann diese APK
  nicht aktualisieren. Deinstalliere diese Vorschau zuerst; Android entfernt dabei
  lokale Einstellungen, konfigurierte Hosts und die Messhistorie.
- **Sprache:** Die Oberfläche in der App ist derzeit nur auf Deutsch verfügbar.
  GitHub-Dokumentation, Projektwebsite und diese Release Notes gibt es auf Englisch
  und Deutsch.
- **Hardware-Prüfung:** APK-Build, Paketmetadaten und Signatur wurden geprüft. Während
  dieses Release-Durchlaufs war kein physischer Android-Handheld für eine Installation
  verbunden.
- Die automatische Sunshine- und Moonlight-Host-Erkennung ist noch nicht implementiert.
- Website-Screenshots sind bereinigte Oberflächenvorschauen und keine echten Geräteaufnahmen.
- Einige WLAN- und Netzwerkwerte können wegen Android-Version, Geräteunterstützung,
  Berechtigungen oder Datenschutzbeschränkungen der Plattform nicht verfügbar sein.
  Die App kennzeichnet nicht verfügbare Werte, statt Messungen zu erfinden.

### Installation oder Update

1. Lade `ThorStreamButler-v0.1.0-alpha.1-debug.apk` aus den Dateien unten herunter.
2. Prüfe sie mit `ThorStreamButler-v0.1.0-alpha.1-debug.apk.sha256`.
3. Erlaube die Installation aus dem verwendeten Browser oder Dateimanager und
   installiere anschließend die APK.
4. Deinstalliere diese Alpha vor einer zukünftigen produktiv signierten stabilen
   Version. Dadurch werden die lokalen App-Daten der Alpha entfernt, da ein Export
   noch nicht verfügbar ist.

Paketname: `de.thorstream.butler`

Minimale Android-Version: Android 9 (API 28)

### Datenschutzhinweis

Thor Stream Butler enthält keine Werbung, kein Tracking und keine Telemetrie.
Netzwerkmessungen und konfigurierte Hosts bleiben lokal auf dem Gerät. Release-Dateien
und öffentliche Screenshots enthalten keine persönlichen Netzwerkdaten.
