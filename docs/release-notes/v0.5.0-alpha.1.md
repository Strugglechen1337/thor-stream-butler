# Thor Stream Butler v0.5.0-alpha.1 — Handheld tools / Handheld-Werkzeuge

## English

### Highlights

- **Quick Settings tile:** the new "Network check" tile in Android's Quick
  Settings shows the most recent stored network quality as its state and
  subtitle. Tapping it opens the app directly into a running network test.
  The tile never measures on its own and reads only the local history.
- **Controller test:** Settings → Appearance and input now opens a live
  diagnostics screen for buttons, D-pad, both analog sticks, and triggers,
  including a list of connected controllers — useful before starting a
  streaming session with a freshly paired gamepad.
- **Device status:** the network test screen shows battery level, charging
  state, and battery temperature, refreshed every ten seconds, with a notice
  when the battery gets hot during long sessions.

### Changes

- Gamepad input is captured only while the controller test is open. BACK and
  all unmapped keys always pass through, so the screen stays escapable by
  touch, keyboard, or the system back gesture.
- Analog axes use a small dead zone and support both common right-stick and
  trigger axis layouts (Z/RZ and RX/RY, LTRIGGER/RTRIGGER and BRAKE/GAS).
- The battery readout uses the permissionless sticky battery broadcast; values
  Android does not report are shown as unavailable instead of being invented.

### Fixes

- Release signature verification invokes `apksigner` from the exact Android
  Build Tools directory instead of relying on the runner's executable search
  path.

### Known limitations

- **Update path:** This build is signed with the permanent production key and
  updates v0.4.0-alpha.1 in place. Debug-signed previews (v0.3 and earlier)
  must be uninstalled first; export the configuration beforehand if needed.
- **Hardware validation:** installation on physical handheld hardware was not
  available during this release run.
- The Quick Settings tile shows the last stored measurement; it does not
  refresh in the background and can therefore be outdated until the next test.
- Automatic background monitoring and notifications for unstable connections
  are not implemented yet.

### Installation

1. Update directly over v0.4.0-alpha.1, or uninstall any older debug-signed
   alpha first (export the configuration beforehand if needed).
2. Download `ThorStreamButler-v0.5.0-alpha.1.apk` and its `.sha256` checksum
   from the release assets.
3. Verify the checksum, allow installation from your file manager or browser,
   and install the APK on Android 9 or newer.

## Deutsch

### Highlights

- **Quick-Settings-Kachel:** Die neue Kachel „Netzwerkcheck“ in den
  Android-Schnelleinstellungen zeigt die zuletzt gespeicherte Netzwerkqualität
  als Status und Untertitel. Ein Tipp öffnet die App direkt mit laufendem
  Netzwerktest. Die Kachel misst nie selbstständig und liest ausschließlich
  die lokale Historie.
- **Controller-Test:** Einstellungen → Darstellung und Bedienung öffnet jetzt
  eine Live-Diagnose für Tasten, Steuerkreuz, beide Analog-Sticks und Trigger,
  inklusive Liste der verbundenen Controller — praktisch, bevor eine
  Streaming-Session mit frisch gekoppeltem Gamepad startet.
- **Gerätestatus:** Der Netzwerktest zeigt Akkustand, Ladezustand und
  Akkutemperatur, alle zehn Sekunden aktualisiert, mit Hinweis, wenn der Akku
  bei langen Sessions heiß wird.

### Änderungen

- Gamepad-Eingaben werden nur bei geöffnetem Controller-Test abgefangen. BACK
  und alle nicht zugeordneten Tasten gehen immer durch; der Bildschirm bleibt
  per Touch, Tastatur oder System-Zurück-Geste verlassbar.
- Analoge Achsen nutzen eine kleine Deadzone und unterstützen die gängigen
  Layouts für rechten Stick und Trigger (Z/RZ und RX/RY, LTRIGGER/RTRIGGER
  und BRAKE/GAS).
- Der Akkustatus nutzt den berechtigungsfreien Sticky-Battery-Broadcast;
  Werte, die Android nicht liefert, werden als nicht verfügbar angezeigt statt
  erfunden.

### Fehlerbehebungen

- Die Release-Signaturprüfung ruft `apksigner` direkt aus dem
  Android-Build-Tools-Verzeichnis auf, statt sich auf den Suchpfad des Runners
  zu verlassen.

### Bekannte Einschränkungen

- **Update-Pfad:** Dieser Build ist mit dem dauerhaften Produktionsschlüssel
  signiert und aktualisiert v0.4.0-alpha.1 direkt. Debug-signierte Vorschauen
  (v0.3 und früher) müssen zuerst deinstalliert werden; exportiere bei Bedarf
  vorher die Konfiguration.
- **Hardware-Prüfung:** Eine Installation auf physischer Handheld-Hardware
  stand für diesen Release-Lauf nicht zur Verfügung.
- Die Quick-Settings-Kachel zeigt die letzte gespeicherte Messung; sie
  aktualisiert sich nicht im Hintergrund und kann bis zum nächsten Test
  veraltet sein.
- Automatische Hintergrundüberwachung und Benachrichtigungen bei instabiler
  Verbindung sind noch nicht umgesetzt.

### Installation

1. Aktualisiere direkt über v0.4.0-alpha.1 oder deinstalliere zuerst eine
   ältere debug-signierte Alpha (bei Bedarf vorher die Konfiguration
   exportieren).
2. Lade `ThorStreamButler-v0.5.0-alpha.1.apk` und die zugehörige
   `.sha256`-Prüfsumme aus den Release-Dateien herunter.
3. Prüfe die Prüfsumme, erlaube die Installation aus Dateimanager oder
   Browser und installiere die APK auf Android 9 oder neuer.
