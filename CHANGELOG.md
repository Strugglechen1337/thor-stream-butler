# Changelog / Änderungsprotokoll

All notable changes are documented in both English and German. Entries in both
language sections must remain equivalent.

Alle wichtigen Änderungen werden auf Englisch und Deutsch dokumentiert. Die
Einträge beider Sprachabschnitte müssen inhaltlich gleichwertig bleiben.

## [Unreleased]

### English

#### Added

- Initial controller-first Android launcher MVP with local network diagnostics,
  host management, Wake-on-LAN, settings, and measurement history.
- Responsive project website with complete English and German variants.
- Bilingual GitHub documentation, community templates, changelog, and release workflow.
- Optional PayPal.me support link and GitHub funding metadata.

#### Known limitations

- No signed public APK release has been published yet.
- Automatic Sunshine and Moonlight discovery is not part of the MVP.
- Real-device screenshots are still planned.

### Deutsch

#### Hinzugefügt

- Erster controller-first Android-Launcher-MVP mit lokaler Netzwerkdiagnose,
  Hostverwaltung, Wake-on-LAN, Einstellungen und Messhistorie.
- Responsive Projektwebsite mit vollständiger englischer und deutscher Variante.
- Zweisprachige GitHub-Dokumentation, Community-Vorlagen, Changelog und Release-Ablauf.
